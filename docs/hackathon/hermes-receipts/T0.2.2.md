## Reproduce locally

1. Install the Hermes CLI and copy `.hermes/config.example.yaml` to
   `.hermes/config.yaml` (git-ignored), filling in real values.
2. Run `pnpm hermes:smoke --no-print-body` to confirm connectivity.
3. Pipe raw session output through the sanitizer and re-run
   `pnpm hermes:verify-receipt` before committing an updated receipt.
